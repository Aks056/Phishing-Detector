üìä Risk Level Calculation

Risk Scoring System

The system calculates risk based on multiple criteria:

Risk Level Determination:
- LOW:    Risk Score < 30
- MEDIUM: Risk Score 30-60
- HIGH:   Risk Score > 60

Final Risk Score = Sum of individual rule threat scores

Multi-Condition Phishing Detection

A URL is flagged as phishing if ANY of these conditions are met:

  • Risk Score ‚â• 30
  • Suspicious subdomains detected (more than 2 dots)
  • HTTPS/SSL certificate is missing or invalid
  • Fake brand detection patterns matched

üöÄ Getting Started

Prerequisites

  • Java Development Kit (JDK) 24 or higher
  • Maven 3.8.1 or higher
  • Modern web browser (Chrome, Firefox, Safari, Edge)

Installation & Setup

# Build the project
mvn clean package -DskipTests

# Run the application
java -jar target/phishing-detector-0.0.1-SNAPSHOT.jar

The application will start on http://localhost:8080
