API

Use the REST endpoint to analyze URLs:

{ "url": "https://example.com" }

Response schema UrlCheckResponse:

  • url: string
  • isPhishing: boolean
  • riskScore: number
  • riskLevel: string
  • detectedThreats: array[string]
  • recommendation: string

OpenAPI / Swagger UI is available at /swagger-ui/index.html once the app is running.
